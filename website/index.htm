Summary

Quake Lenses is a proof-of-concept for trying out new fields of view (FOVs) in 3D games. It allows you to play Quake using a variety of different lenses, and it even allows you to make your own.

This is a late successor to Fisheye Quake. It uses the TyrQuake source port for its code base and embeds the Lua interpreter for rapid lens prototyping.

Installation

Quake (or the free Quake demo) is required. Once you have it, you can download Quake Lenses here. Extract the files into your Quake directory, and run quakelens.exe to play.

Tutorial

Available Commands

  • lens : change the lens
  • hfov : specify FOV in horizontal degrees
  • vfov : specify FOV in vertical degrees
  • fit : make the lens fit the screen
  • hfit : fit horizontally only
  • vfit : fit vertically only

Lens Files

In the lenses folder of your quake directory, you will see a collection of files. Each file is a Lua script used by the game to construct a lens. If you're interested, you can create your own and place it in this folder.

Using a Lens in game

First, open quakelens.exe, and start a game. To change your lens, you must first pull down the console by pressing the tilde (~) key. If you wish to use the "panini" lens, type lens panini. The game will then look for a file called "panini.lua" in the lenses folder and it will load it.

Adjusting the Lens Zoom

There are two ways to adjust the zoom of your lens. If one doesn't work, you can try the other.

FOV

You can zoom in/out by specifying an FOV angle. Type hfov 180 if you want your screen to horizontally cover 180 degrees. Similarly, you can type vfov 90 to cover 90 degrees vertically.

Fitting

If the lens does not produce a picture of infinite size, you can fit it to your screen by typing fit. Alternately, if you just want the picture to fit the horizontal or vertical bounds of the screen, you can type hfit or vfit, respectively.
