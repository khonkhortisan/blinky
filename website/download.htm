Compiling for Windows

This requires MinGW. Also, install LuaJIT into the MinGW file system. Type: 
> make prepare
> make tyr-quake.exe

Compiling for Linux

This requires LuaJIT. To compile, type: 
> make prepare
> make tyr-quake
